## Overview

The TMC2209 is a single-chip bipolar stepper motor driver integrating two full H-bridges with all power MOSFETs, charge pump, current sensing logic, and motion-control algorithms. It targets applications up to 2A RMS per coil (2.8A peak) at supply voltages from 4.75V to 29V. The IC operates in standalone Step/Dir mode or via single-wire UART, supporting up to four nodes on one bus. StealthChop2 voltage-mode PWM delivers near-silent operation at low velocities; SpreadCycle current-mode chopper covers higher-speed ranges. StallGuard4 provides sensorless stall detection enabling encoderless homing. MicroPlyer interpolates step inputs to 256 microsteps regardless of MCU resolution. The package is a 28-pin QFN 5×5 mm (0.5 mm pitch) with an exposed thermal slug.

## Interfaces

### Step / Dir (Standalone)

Standard two-wire pulse interface. Microstep resolution set at power-on from MS1/MS2 pins; direction from DIR; ENN active-high disables the output stage. No MCU communication required. PDN_UART pulled low in standalone enables the automatic standstill current reduction via the pin.

### UART (Single-Wire, Half-Duplex)

- **Physical:** Single wire on PDN_UART; open-drain from master side; internal 50 kΩ pull-up to VCC_IO inside IC
- **Baud rate:** 9,600 – 500 kBaud (auto-detected via sync nibble on each datagram); max = f_CLK / 16
- **Frame format:** 8 data bits, no parity, 1 stop bit (standard UART)
- **Write datagram:** 8 bytes — sync nibble + node addr + reg addr (write bit set) + 32-bit data + CRC8-ATM
- **Read datagram:** 4-byte request → 8-byte reply; reply node address = 0xFF
- **Multi-node:** Up to 4 nodes on one bus; address 0–3 set by MS1/MS2 at power-on
- **SENDDELAY:** Must be ≥ 2 (in NODECONF, reg 0x03) when using multiple nodes to prevent bus collisions
- **Required config:** Set `GCONF.pdn_disable = 1` when using UART (disables PDN pin auto-powerdown function)
- **Write verification:** Read IFCNT (reg 0x02); value increments 0–255 wrapping on each successful write
- **mstep_reg_select:** Set `GCONF.mstep_reg_select = 1` to set microstep resolution via MRES in CHOPCONF rather than MS1/MS2 pins

## Motion Features

### StealthChop2

Voltage-mode PWM chopper for near-silent operation at low speeds. Two-stage auto-tuning: AT#1 calibrates PWM offset at standstill (≤ 130 ms); AT#2 continuously adapts gradient during motion. Key PWMCONF settings (reset default `0xC10D0024`):

- `pwm_autoscale = 1` — enables automatic amplitude control (default on)
- `pwm_autograd = 1` — enables automatic gradient adaptation (default on)
- `PWM_OFS` — base PWM offset (auto-tuned)
- `PWM_GRAD` — PWM amplitude gradient (auto-tuned)
- `pwm_freq` — PWM frequency selection; at 12 MHz internal clock with `FREQ = %01` (default): ~35.1 kHz
- `FREEWHEEL` (PWMCONF bits 21:20): 0 = normal, 1 = freewheeling, 2 = LS coil short, 3 = HS coil short (active when IHOLD = 0)

StealthChop is active when `TSTEP ≥ TPWMTHRS`. Above this threshold (higher velocity), the driver switches to SpreadCycle.

### SpreadCycle

Cycle-by-cycle current-control chopper with zero-crossing detection. Enabled by pulling SPREAD pin high or setting `GCONF.en_SpreadCycle = 1`. Chopper frequency target: 16–30 kHz. Configured via CHOPCONF (reset default `0x10000053`):

- `TOFF` — minimum off-time; must be ≥ 1 to enable driver
- `TBL` — blanking time
- `HSTRT` — hysteresis start value
- `HEND` — hysteresis end value
- `vsense` — 0 = VFS 325 mV; 1 = VFS 180 mV (higher sensitivity, needed for lower current)

### MicroPlyer

Hardware interpolation from any MRES setting to 256 microsteps. Enabled by `CHOPCONF.intpol = 1` (default). Requires jitter-free STEP input. At 12 MHz internal clock, step frequencies below ~12 Hz are interpreted as standstill by MicroPlyer. For full 256-microstep native mode, set `MRES = 0` in CHOPCONF and `intpol = 0`.

### StallGuard4

Sensorless stall detection operating exclusively in StealthChop mode. DIAG output asserts when SG_RESULT ≤ SGTHRS × 2 and velocity is in the CoolStep/StallGuard active window (`TCOOLTHRS ≥ TSTEP > TPWMTHRS`). Updated once per electrical fullstep.

- `SGTHRS` (reg 0x40, write-only): stall threshold 0–255
- `SG_RESULT` (reg 0x41, read-only): 10-bit result 0–510; 0 = stall/high load, 510 = no load
- Calibration: find maximum load SG_RESULT at target velocity, set SGTHRS ≈ 50% of that value
- Disable CoolStep (`SEMIN = 0`) when using StallGuard for sensorless homing to avoid interference

### CoolStep

Adaptive current scaling based on StallGuard4 readings. Reduces current at light load (saving up to 75% power); increases back to IRUN under load. Configured via COOLCONF (reg 0x42):

- `SEMIN` (bits 3:0): minimum StallGuard reading to enable CoolStep; 0 = CoolStep disabled
- `SEMAX` (bits 11:8): upper threshold for current reduction
- `SEUP` (bits 6:5): current increment steps
- `SEDN` (bits 14:13): current decrement steps
- `SEIMIN` (bit 15): minimum current floor — 0 = 1/2 IRUN (requires IRUN ≥ 10); 1 = 1/4 IRUN (requires IRUN ≥ 20)

### VACTUAL — Internal Step Generator

`VACTUAL` (reg 0x22, signed 24-bit): sets internal step generation velocity, bypassing STEP pin. Motor direction from sign. Set to 0 to return to STEP/DIR input mode.

Velocity formula: `v [Hz] = VACTUAL × f_CLK / 2^24` ≈ `VACTUAL × 0.715 Hz` with internal 12 MHz clock.

## Current Setting

### External Sense Resistors (Recommended)

Connect sense resistors between motor coil outputs and BRA/BRB pins. Full-scale current set by VREF voltage and vsense bit:

| vsense | VFS | 100 mΩ → I_RMS | 150 mΩ → I_RMS |
|---|---|---|---|
| 0 (default) | 325 mV | ~1.92 A | ~1.35 A |
| 1 | 180 mV | ~1.06 A | ~0.71 A |

VREF range: 0.5–2.4 V for linear scaling. Open-circuit VREF = 1.8 V (73% scale). Full-scale = 2.5 V. At IRUN = 31 (register full scale), actual current = VFS / (√2 × R_sense).

### Internal RDS(ON) Current Sensing

Set `GCONF.internal_Rsense = 1`; tie BRA and BRB to GND; connect a resistor (RREF) from 5VOUT to VREF to set the scale. Example values:

| RREF | I_RMS |
|---|---|
| 8.2 kΩ | ~1.4 A |
| 12 kΩ | ~1.0 A |

Best accuracy range: 200 mA – 1.4 A RMS. Not recommended for > 1.4 A.

### IHOLD / IRUN Registers

`IHOLD_IRUN` (reg 0x10, write-only):
- `IHOLD` (bits 4:0): standstill current, 0–31; OTP default = 16
- `IRUN` (bits 12:8): run current, 0–31; OTP default = 31
- `IHOLDDELAY` (bits 19:16): delay from run to hold current (0 = immediate; each step = 2^18 / f_CLK)

`TPOWERDOWN` (reg 0x11): standstill delay before automatic current reduction; default = 20 (range 0–255 × 2^18 tCLK, ~0–5.6 s at 12 MHz).

## Protection Features

All fault status flags are in `DRV_STATUS` (reg 0x6F, read-only):

| Flag | Condition | Behavior |
|---|---|---|
| `ot` | Junction ≥ 143 °C (default) | Driver disabled; auto-recovers below OTPW threshold |
| `otpw` | Junction ≥ 120 °C (default) | Warning flag only; can trigger INDEX output |
| `s2ga` / `s2gb` | Short-to-GND, coil A/B | Retries 3× then disables; VS−Vout threshold ~2.5 V typ |
| `s2vsa` / `s2vsb` | Short-to-VS, coil A/B | Detected at ~2 V above GND; disables bridge |
| `ola` / `olb` | Open load, coil A/B | Informational only; no action; check in SpreadCycle at low velocity |
| `stst` | Motor at standstill | Informational |
| `stealth` | StealthChop active | Informational |
| `CS_ACTUAL` (bits 12:8) | Actual current scale 0–31 | Read-only, reflects CoolStep/autoscale result |

Additional in `GSTAT` (reg 0x01, read + write-to-clear):
- `reset`: set on power-on or watchdog reset
- `drv_err`: set when driver error (short/OT) occurred; cleared by reading GSTAT
- `uv_cp`: charge pump undervoltage (bridge disabled while asserted)

Short detector propagation delay: 1.3 µs typ. Driver can be re-enabled after fault by toggling ENN or setting TOFF = 0 then restoring.

## PCB Layout Notes

- **VS decoupling:** ≥ 100 µF low-ESR electrolytic close to the IC, plus 100 nF ceramic; minimize loop area
- **5VOUT:** 2.2–4.7 µF ceramic directly to die GND pad; place within 1 mm of pin 8
- **Charge pump:** 22 nF 50V ceramic between CPO (pin 4) and CPI (pin 5); 100 nF ceramic between VCP (pin 6) and VS
- **Thermal slug:** Must connect to GND copper plane with ≥ 4 thermal vias; θ_JC = 6 K/W; θ_JA = 30 K/W (2s2p multilayer)
- **Sense resistors:** Short, symmetric, low-inductance traces; no shared GND return between phase A and phase B resistors
- **VS slew rate:** Keep dV/dt < 1 V/µs on VS rail to protect charge pump
- **External clock:** Use 8–13.4 MHz nominal for best accuracy; tie CLK directly to GND (not via resistor) for internal oscillator

## Typical Applications

- 3D printers (Marlin, Klipper) — most widely deployed Trinamic driver in this space
- Light CNC machines (laser engravers, PCB routers)
- Office automation (paper transport, label printers)
- Textile machinery
- Camera gimbals, sliders, and focus actuators
- Any 12–24 V motion system where silent operation or sensorless homing is required

## Selection Notes

The TMC2209 is appropriate when:

- Coil current ≤ 1.4 A RMS continuous (or ≤ 2.0 A with duty cycling)
- Supply voltage 5–29 V
- UART-only (no SPI) is acceptable
- StealthChop silence and/or StallGuard4 sensorless homing are required
- Low component count and low BOM cost are priorities

**Limitations vs. alternatives:**
- No SPI interface → use [[TMC2240]] (SPI + UART, 36V, 2.1A RMS, lossless sensing) or [[TMC5160A]] (SPI + UART, external FETs, >20A possible)
- No encoder input → use [[TMC4671-LA]] for closed-loop servo control
- StallGuard4 only works in StealthChop; no support in SpreadCycle-only mode
- Internal RDS(ON) sensing accuracy degrades above 1.4 A RMS; external sense resistors recommended for precision applications
- All register state is lost on STDBY pin assertion (hardware standby resets the IC)
