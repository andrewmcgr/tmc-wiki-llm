This directory contains knowledge articles on stepper motor driver chips — their capabilities, configuration, and comparative analysis.

## Pages

- [[TMC2209]] — 2A RMS, 29V, UART, StealthChop2/SpreadCycle/StallGuard4/CoolStep. The go-to driver for 3D printers.
- [[TMC2240]] — 2.1A RMS, 36V, SPI+UART, lossless current sensing, encoder input. Next-gen integrated driver.
- [[TMC5160A]] — External FETs, 20A+, 60V, SPI+UART, SixPoint ramp generator. High-power applications.
- [[TMC4671-LA]] — Hardware FOC servo controller, BLDC/stepper/DC/voice coil, external power stage (e.g., TMC6100). Not a stepper driver — a motion controller IC.
- [[TMC-Driver-Comparison]] — Side-by-side comparison of all four ICs: specs, features, interfaces, protection, and selection guidance.
